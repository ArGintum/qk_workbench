This folder contains data and code for our experiments form ... paper (published at EMNLP 2025). 

**Structure:**
- `data`  --- storage folder for the datasets
  - `generate_data.ipynb` --- jupyter-notebook with preset commands to generate datasplits that were used in our experiments in the paper
- `cache` --- storage folder for output files (i.e., raw QK-scores, baseline predictions, etc.)
- `prontoqa_example.ipynb` --- jupyter-notebook with code for reproducing our experiments on ProntoQA-OOD dataset
